    # Vuepress搭建个人博客及GitHub Pages部署

    # 一、起步

    # 1、创建项目目录

    mkdir vuepress

cd vuepress

npm init //初始化npm包

    # 2、安装vuepress

    npm i  -D vuepress

    # 3、创建文档

    mkdir docs

echo '# Hello VuePress' > docs/README.md

    # 4、配置文件package.json

    "scripts": {`  

  `"docs:dev": "vuepress dev docs", 	//	启动本地服务`

  `"docs:build": "vuepress build docs"	//	项目打包`

 `}

    # 二、配置

    # 1、在docs目录下添加.vuepress目录

    cd docs
mkdir .vuepress

    所有 VuePress 相关的文件都会被放在这里。在 .vuepress 文件夹下添加 config.js，配置网站的标题和描述

    此时目录结构:

    |-- docs',
    |-- README.md',
    |-- .vuepress',
        |-- config.js'

    # 2、配置config.js

    module.exports = { 
	title: '我的博客',   //标题
	description: 'XXX'  //描述
	}

    # 3、添加导航栏

    module.exports = {
    title: '我的博客',
    description: 'XXX',
    themeConfig:{
        nav:[
            { text:'首页',link:'/'},
            { 
                text:'hannz的博客',
                items:[
                    { text: "掘金", link: "https://juejin.cn/user/3558485311363677" },
                    { text: "Github", link: "https://github.com/han818" },
                    { text: "码云", link: "https://gitee.com/han-zzgo" }
                ]
        }
    ],
   
    },
   
}

    # 4、添加侧边栏

    在docs目录下新增 handbook目录，在里面新增md文档

    |-- docs',
    |-- README.md',
    |-- .vuepress',
        |-- config.js'
    |-- handbook
        |-- 1.md 
        |-- 2.md

    config.js配置如下:

    module.exports = {
    title: '我的博客',
    description: 'XXX',
    themeConfig:{
        nav:[
            { text:'首页',link:'/'},
            { 
                text:'hannz的博客',
                items:[
                    { text: "掘金", link: "https://juejin.cn/user/3558485311363677" },
                    { text: "Github", link: "https://github.com/han818" },
                    { text: "码云", link: "https://gitee.com/han-zzgo" }
                ]
        }
    ],
    sidebar:[
        {
            title:'A',
            path:'/',
            collapsable:false, //是否折叠
            children:[{title:'博客简介',path:'/'}],
        },
        {
            title:'B',
            path:'/handbook/1',
            collapsable:true,
            children:[
                {title:'第一篇',path:'/handbook/1'},
                {title:'第二篇',path:'/handbook/2'},
            ],
        },
    ],
    },
  
}

    # 5、主题插件

    安装vuepress-theme-reco插件

    npm install vuepress-theme-reco --save-dev

    在config.js中引用

    module.exports = {
  theme: 'reco',
  ...
}

    # 6、添加文章信息

    在md文档中添加:

    ---
title: 2.
author: hannzz
date: '2022-9-26'
---

    # 7、设置语言

    module.exports = {
  ...
  locales: {
    '/': {
      lang: 'zh-CN'
    }
  },
  ...
}

    可以发现 我们写入的时间格式为2022-9-26 ，但显示的是2022/9/26。 这是因为 VuePress 默认的 lang 为 en-US，我们修改为zh-CN

    # 三、部署到Github Page

    # 1、设置仓库

    在GitHub上新建仓库，如:interview

    需在config.js中配置base路径

    module.exports = {
  // 和仓库名相同
  base: '/interview/',
  ...
}

    module.exports = {
    title: '我的博客',
    description: 'XXX',
    base: '/interview/',
    themeConfig:{
        nav:[
            { text:'首页',link:'/'},
            { 
                text:'hannz的博客',
                items:[
                    { text: "掘金", link: "https://juejin.cn/user/3558485311363677" },
                    { text: "Github", link: "https://github.com/han818" },
                    { text: "码云", link: "https://gitee.com/han-zzgo" }
                ]
        }
    ],
    sidebar:[
        {
            title:'欢迎学习',
            path:'/',
            collapsable:false, //是否折叠
            children:[{title:'博客简介',path:'/'}],
        },
        {
            title:'基础篇',
            path:'/handbook/1',
            collapsable:true,
            children:[
                {title:'第一篇',path:'/handbook/1'},
                {title:'第二篇',path:'/handbook/2'},
            ],
        },
    ],
    },
    theme: 'reco',
    locales: {
        '/': {
            lang: 'zh-CN'
        }
    },
    subSidebar: 'auto',
}

    # 2、创建并执行脚本

    在项目根目录下创建脚本文件deploy.sh，写入如下:

    #!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
npm run docs:build

# 进入生成的文件夹
cd docs/.vuepress/dist

git init
git add -A
git commit -m 'deploy'

# 发布
# git push -f git@github.com:XXX/XXX.git master:gh-pages
git push -f git@github.com:han818/interview.git master:gh-pages

cd -

    配置完脚本文件后，在项目根目录 打开gitbash

    输入sh deploy.sh执行脚本

    cd vuepress
sh deploy.sh

    # 3、查看

    在github仓库中的setting->Pages中能找到博客地址

    地址:https://han818.github.io/interview/

    以上即为vuepress+github pages搭建博客的大概流程
