    # 1、 js中获取dom节点的方法(函数)?

    getElementById getElementsByTagName getElementsByClassName

    # 2、 js中通过节点关系获取节点的属性有哪些?

    firstElementChild lastElementChild childNodes children nextElementSibling previousElementSibling parentNode

    # 3、 js中如何获取节点的属性?(内置属性和方法分别说明)

    className value style dom.属性名 dom.getAttribute( )

    # 4、js中如何设置节点的属性?(内置属性和方法分别说明)

    dom.属性名="" dom.setAttribute('属性名','value')

    # 5、 js中如何删除节点属性?

    dom.属性名='' (空值) dom.removeAttribute('属性名')

    # 6、 js中如何获取节点样式?(两种方式分别说明)

    dom.style.样式名 (只能获取行内样式) offsetWidth... scrollWidth... clientWidth... getComputedStyle(dom).样式名

    # 7、 js中如何绑定节点的事件?(两种方式分别说明)

    dom.onclick = function(){} dom只能绑定一个事件

    dom.addEventListener('click',function(){}) dom可以绑定多个事件

    移除事件监听事件 dom.removeEventListener('click',fun)

    # 8、 js中dom事件对象包含什么内容?

    事件类型 事件源 鼠标坐标 键盘按键 还有一些事件方法

    # 9、 js事件中如何阻止浏览器默认行为?10，js事件中如何阻止事件冒泡?

    event.preventDefault()

    # 10、js事件中如何阻止事件冒泡?

    event.stopPropagation

    # 11、css中基础选择器有哪些?

    *(通配符) 标签选择器 类选择器 属性选择器 id选择器

    # 12、css中派生(组合)选择器有哪些?

    标签指定式选择器 后代选择器(以空格分隔) 子元素选择器(以大于号分隔）> 相邻兄弟选择器（以加号分隔）+ 普通兄弟选择器（以破折号分隔）~

    # 13、css中如何实现颜色透明?

    rgba(red,green,blue,alpha) transparent

    # 14、css中如何实现过渡效果?

    transition:all 1s ease 1s

    # 15、css中如何设置背景尺寸?

    background-size:width|height
